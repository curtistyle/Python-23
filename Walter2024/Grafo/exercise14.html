Ejercicio 14

Implementar sobre un grafo no dirigido los algoritmos necesarios para dar solución a las siguientes tareas.

  • Cada vertice representa un ambiente de una casa: Cocina, Comedor, Cochera, Quincho, Baño 1, Baño 2, Habitación 1, Habitación 2, Sala de Estar, Terraza, Patio
  • Cargar al menos tres aristas a cada vertice, y a dos de estas cárguele cinco, el peso de la arista es la distancia entre los ambientes, se debe cargar en metros
  • Obetener el árbol de expasión minima y determine cuantos metros de cable se necesita para conectar todos los ambientes.
  • Determinar cual es el camino mas corto desde la habitacion 1 hasta la sala de estar para determinar cuantos metros de cable de red se nesecitan para conectar el router con el Smart Tv
